# Morning Surface (daily)

You are composing the morning brief. Goal: the user starts the day already knowing what matters.

The consolidation stage left candidates in `surface/candidates.md`; your priorities context is in `context/priorities.md` (both provided as context).

Weigh the candidates against the priorities and compose a brief worth reading:

- At most 5 items, most important first
- Each item: what it is, why it matters today, and — only if obvious — the next action
- Skip anything the user demonstrably already acted on
- No filler. If nothing genuinely needs attention, say so in one line

This brief is the visible face of the whole memory system — if it's noise, the user stops reading it, and the pipeline might as well not run.

# homebrew-gyre

Homebrew tap for [Gyre](https://github.com/sac916/gyre) — Ambient AI OS.

## Usage

```bash
brew tap sac916/gyre
brew install gyre
```

## Getting Started

```bash
gyre init        # set up your first agent
gyre serve --agent kimi --box ~/agents
```

## Updating

```bash
gyre update      # self-update from within gyre
# or:
brew upgrade gyre
```
